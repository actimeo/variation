Description

Ce script permet de fusionner des données dans un formulaire PDF. En partant d'un modèle contenant des champs textes, il est possible d'injecter les valeurs de deux manières différentes :
  • à partir d'un tableau PHP
  • à partir d'un fichier FDF
Le document final est renvoyé par la méthode Output(), identique à celle de FPDF.

Remarque : si votre PDF modèle n'est pas compatible avec le script, vous pouvez le retraiter avec pdftk de cette manière :

pdftk modele.pdf output modele2.pdf

Et réessayez avec modele2.pdf.
